Nome do projeto Descrição Ano

Este projeto foi criado para reunir minhas principais informações acadêmicas e profissionais em um único site. Nele, trabalhei estruturação semântica em HTML, estilização com CSS, responsividade, animações suaves e organização visual das seções como início, sobre mim, conhecimentos, projetos e contato.

Projeto voltado para a criação de um layout moderno para uma loja. O foco principal foi trabalhar estrutura visual, organização de conteúdo e harmonia entre tipografia, cores e espaçamentos.

Conjunto de atividades práticas desenvolvidas durante a disciplina de desenvolvimento web. Esses exercícios ajudaram a fortalecer minha base em estruturação de páginas HTML e estilização com CSS.
